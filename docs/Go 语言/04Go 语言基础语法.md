### Go 标记
Go 程序可以由多个标记组成，可以是`关键字，标识符，常量，字符串，符号`。如以下 GO 语句由 6 个标记组成：
```
fmt.Println("Hello, World!")
```
6 个标记是(每行一个)：  
```
1. fmt
2. .
3. Println
4. (
5. "Hello, World!"
6. )
```
