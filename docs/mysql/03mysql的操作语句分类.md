title: mysql的操作语句分类
description: mysql的操作语句分类
tags: [mysql]
---

1. **DDL 数据定义语言(Data Definition Language)**  
    例如: 建库、建表
2. **DML 数据操作语言(Data Manipulation Language)**  
    例如：增删改
3. **DQL 数据查询语言(Data Query Language)**     
    例如：查询
4. **DCL 数据控制语言(Data Control Language)**     
    例如：对用户权限进行设置 GRANT DENY
