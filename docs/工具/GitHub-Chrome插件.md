#### 1. Octotree - GitHub code tree

![介绍](/img/工具/octotree-chrome.png)

![截图](/img/工具/octotree.png)

增强 GitHub 代码审查和探索的浏览器扩展。  

特征  
* 类似 IDE 的快速代码树  
* 以树形格式快速搜索  
* 书签回购、问题、PR、文件  
* 支持 GitHub 主题  
* 支持私有仓库  
* 高性能，适用于任何大小的存储库
