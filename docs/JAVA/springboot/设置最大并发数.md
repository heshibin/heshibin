springBoot内置Tomcat，在默认设置中，Tomcat的最大线程数是200，最大连接数是10000。支持的并发量是指连接数，也就是说 支持最大并发量是一万(默认)。

```yml
server:
  tomcat:
    max-threads: 1000
    max-connections: 20000
```
