**简介**

对网站、App等进行业务数据采集，包括用户行为数据及其他实际需要的数据进行采集上报。

**目的**

方便分析用户行为、提高用户体验，通过这些采集的数据就可以进行分析并辅助公司企业做决策。

**方式**

代码埋点、可视化埋点、全埋点
