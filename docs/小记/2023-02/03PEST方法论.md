#### 什么是PEST方法论？
1. 指的是政治（Political）、经济（Economic）、社会（Social）和技术（Technological）。
2. 政治环境主要是看我们国家现在是否鼓励相关的业务。
3. 经济环境又可以分为宏观经济和微观经济，包括居民消费水平、产业结构。
4. 社会环境则是说跟社会的风俗习惯是否吻合。
5. 技术环境当然就是说的我们的技术实例（ASML光刻机）。
